# Лабораторная работа №3: MPI-параллельное умножение матриц

## Задание

Модифицировать программу из лабораторной работы №1 для параллельной работы по технологии MPI.

### Экспериментальные параметры

| Параметр | Значения |
|----------|----------|
| Размеры матриц | 200, 400, 800, 1200, 1600, 2000 |
| Количество процессов | 1, 2, 4, 8 |
| Количество запусков | 3 (усреднение) |

## Структура проекта

- `matrix_mpi.cpp` — MPI-версия программы
- `CMakeLists.txt` — сборка проекта
- `generate_matrix.py` — генерация матриц
- `verify.py` — верификация
- `run_experiments.py` — автоматизация экспериментов
- `analyze_results.py` — анализ и графики
- `run_job.slurm` — скрипт для запуска на кластере
