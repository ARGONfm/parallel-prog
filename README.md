# Лабораторная работа №1: Параллельное умножение матриц с использованием OpenMP

## Задание
- Перемножение двух квадратных матриц
- Вход: файлы matrixA.txt, matrixB.txt
- Выход: result.txt + время выполнения
- Параллелизация с OpenMP
- Верификация результата с помощью NumPy (verify.py)
- Проведение экспериментов для матриц размером 200, 400, 800, 1200, 1600, 2000
- Исследование ускорения при использовании 1, 2, 4, 8 потоков

## Технологии
- C++17
- OpenMP
- CMake
- Python 3.x (pandas, matplotlib, numpy)

## Структура проекта
- `matrix_mult.cpp` — основная программа с OpenMP параллелизацией
- `CMakeLists.txt` — сборка проекта
- `generate_matrix.py` — генерация тестовых матриц
- `verify.py` — проверка корректности результата
- `run_experiments.py` — скрипт для автоматического запуска экспериментов
- `plot_results.py` — построение графиков по результатам замеров
- `REPORT.md` — отчёт с замерами, таблицами и выводами
