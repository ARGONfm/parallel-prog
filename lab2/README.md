# Лабораторная работа №2: Исследование масштабируемости OpenMP

## Задание

Модифицировать программу из лабораторной работы №1 для исследования масштабируемости параллельных вычислений с использованием OpenMP.

### Экспериментальные параметры

| Параметр | Значения |
|----------|----------|
| Размеры матриц | 200, 400, 800, 1200, 1600, 2000 |
| Количество потоков | 1, 2, 4, 8 |
| Количество запусков | 3 (усреднение) |

### Метрики

- **Время выполнения** — среднее время умножения матриц
- **Ускорение** — S(p) = T(1) / T(p)
- **Эффективность** — E(p) = S(p) / p

---

## Структура проекта

- `matrix_mult.cpp` — основная программа умножения матриц с OpenMP
- `CMakeLists.txt` — сборка проекта
- `generate_matrix.py` — генерация тестовых матриц
- `verify.py` — проверка корректности результата
- `run_experiments.py` — автоматический запуск экспериментов
- `analyze_results.py` — анализ результатов и построение графиков
- `experiment_results.csv` — результаты экспериментов (генерируется)
- `plots/` — папка с графиками (генерируется)
- `REPORT.md` — отчёт с замерами и выводами
